Fahrenheit et Celsius

ficher : tconvert.c

3 étapes : - °C vers °F / °F vers °C
           - Entrer la température.
           - Affichage de la conversion.

Nécessaire :
Formule de conversion avec des float. 
F = (9/5) * C + 32
C = (F - 32)*5/9

%.2f pour afficher des nombres floattans avec 2 zero après la virgule.
